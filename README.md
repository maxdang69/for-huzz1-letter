# Романтическая страница — деплой и редактирование

Чистый HTML/CSS/JS, без сборки, без зависимостей. Готово для GitHub Pages.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub, залей в него всё содержимое этой папки
   (`index.html`, `style.css`, `script.js`, `photos/`, `music/`, `assets/`)
   в корень репозитория.
2. В репозитории: **Settings → Pages → Source** → выбери ветку (обычно `main`)
   и папку `/ (root)`.
3. Через минуту-две сайт будет доступен по адресу вида
   `https://<твой-юзернейм>.github.io/<репозиторий>/`.

Никакого билда не нужно — статика отдаётся как есть.

## Что и где менять

| Что меняешь | Файл | Где именно |
|---|---|---|
| Текст письма | `script.js` | переменная `letterText` вверху файла |
| Список фото | `script.js` | массив `photos` (пути относительно корня) |
| Сами фото | `photos/` | замени `1.jpg`, `2.jpg`, `3.jpg` своими, или добавь новые и пропиши их в `photos[]` |
| Музыка | `music/` | положи файл и поправь `MUSIC_SRC` в `script.js`, если имя другое |
| Громкость по умолчанию | `script.js` | `DEFAULT_VOLUME` (сейчас `0.35`) |
| Цвета темы | `style.css` | блок `:root { ... }` в самом верху |

## Важно про музыку

Сейчас в `music/` лежит только `README.txt` — самого mp3-файла с
LeeHi — ONLY там нет, это чужой авторский трек, я не стал его подкладывать.
Добавь файл сам и, если назовёшь иначе, поправь `MUSIC_SRC` в `script.js`.
Если файла не будет вообще — сайт не сломается, просто не будет звука.

## Структура

```
index.html      — разметка (конверт, книжка, ползунок громкости)
style.css       — все стили и анимации, цвета вынесены в CSS-переменные
script.js       — вся логика: конверт, телепорт кнопки, листание книжки, аудио
photos/         — фото для книжки (1.jpg, 2.jpg, 3.jpg — замени своими)
music/          — сюда кладёшь mp3
assets/         — на будущее, если понадобятся доп. картинки/иконки
```

## Технические заметки

- Шрифты (`Press Start 2P`, `Caveat`, `Nunito`) подключены через Google Fonts
  CDN — нужен интернет у того, кто открывает страницу. Если нужна полная
  офлайн-версия — скажи, заменю на локальные `@font-face`.
- Анимация перелистывания — CSS `rotateY` с `perspective`, без библиотек.
- Кнопка «Не открывать» телепортируется по клику в случайную точку экрана,
  с проверкой, чтобы не перекрыть конверт/кнопку «Открыть»/книжку и не вылезти
  за края экрана.
- Протестировано на вёрстку в диапазоне 360–430px и на десктопе; сам JS-флоу
  (открытие конверта → книжка → листание → аудио) проверен логически,
  живого браузера в моей среде для скриншота нет — рекомендую быстро
  открыть `index.html` локально перед деплоем и проверить вживую.
