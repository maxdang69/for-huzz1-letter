Сюда кладёшь свои фото.

Сейчас лежат 3 плейсхолдера (1.jpg, 2.jpg, 3.jpg) — просто чтобы книжка
сразу работала. Замени их своими файлами с такими же именами,
либо добавь новые и пропиши пути в массиве `photos` в script.js:

  const photos = [
    "photos/1.jpg",
    "photos/2.jpg",
    "photos/3.jpg",
    "photos/4.jpg"   <- например, новое фото
  ];

Поддерживаются форматы: .jpg, .jpeg, .png, .webp
Порядок показа в книжке = порядок в массиве, GitHub Pages не отдаёт
список файлов папки сам, поэтому путь нужно прописывать вручную.
